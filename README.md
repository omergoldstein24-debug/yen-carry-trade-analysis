# Yen Carry-Trade Analysis (2015–2025)

An end-to-end Python project that quantifies the relationship between the
US–Japan 10-year interest-rate differential and the USD/JPY exchange rate,
and simulates the P&L of a classic yen carry trade over the last decade.

> **TL;DR** — Over 2015–2025 a $100 carry trade (borrow JPY at the 10Y JGB
> yield, invest in US 10Y Treasuries, mark-to-market monthly through the FX
> rate) grew to roughly **$169**, but only after spending five years
> underwater. The rate differential is a weak monthly predictor
> (r ≈ 0.05) — most of the gains came from a regime shift after 2022 when
> the differential exceeded 3.5%.

## Quickstart

```bash
# 1. Clone the repo and create a virtual environment
python -m venv .venv
source .venv/bin/activate        # Windows: .venv\Scripts\activate

# 2. Install dependencies
pip install -r requirements.txt

# 3. Rebuild the clean dataset, print the stats, and regenerate every figure
python main.py
```

The driver script writes:
- `data/processed/usd_jpy_interest_rates_clean.csv`
- `figures/01_japan_10y_yield.png`
- `figures/02_rate_differential_scatter.png`
- `figures/03_jpy_weakening_by_bucket.png`
- `figures/04_carry_trade_cumulative.png`
- `figures/05_carry_trade_diagram.png`

To run a single stage, invoke its module directly:

```bash
python -m src.data_pipeline     # rebuild only the clean CSV
python -m src.analysis          # print regression + bucket summary
python -m src.visualization     # regenerate the figures
```

## Data sources

| Series           | Frequency | Source                                            | Notes                                |
|------------------|-----------|---------------------------------------------------|--------------------------------------|
| Japan 10Y yield  | monthly   | FRED — `IRLTLT01JPM156N`                          | Long-term government bond yield, %   |
| US 10Y yield     | monthly   | FRED — `GS10`                                     | 10Y Treasury constant maturity, %    |
| USD/JPY          | monthly   | Investing.com — historical data export             | End-of-month close                    |

All three series cover **January 2015 → December 2025** (132 monthly obs.).

---

## Method

1. **Data pipeline** (`src/data_pipeline.py`)
   Loads each source, normalises column names and dates, and merges
   the three series on a common month-start timestamp. A derived
   `rate_differential = us_10y_yield − japan_10y_yield` column is added.

2. **Analysis** (`src/analysis.py`)
   - Monthly USD/JPY returns via `pct_change`.
   - Pearson correlation and OLS slope between the differential and the
     next month's FX move.
   - A bucketed-frequency view: in what share of months did JPY weaken,
     grouped by the differential regime (Low / Medium / High / Very High)?
   - A simple carry-trade simulation:
     `r_t = (1 + diff_t/12/100) · (1 + ΔUSDJPY_t) − 1`.

3. **Visualisation** (`src/visualization.py`)
   Reproduces the five figures shown in the write-up using matplotlib
   with no external theme dependencies.

## Key findings

- The US-minus-Japan 10Y yield differential widened from ≈1.6% in 2015 to a
  peak of ≈3.8% in 2024 — but the Japan side only started moving meaningfully
  after 2022.
- Cross-sectionally, **a bigger differential does not reliably mean a weaker
  yen on a monthly basis** (r ≈ 0.05). The month-to-month relationship is
  dominated by noise.
- **Regime matters.** When the differential exceeds 3.5%, USD/JPY rose in
  **71%** of months — well above the 50% random baseline. Below 3.5% the
  signal is indistinguishable from a coin flip.
- A $100 carry portfolio spent roughly five years underwater (trough ≈ $87
  in late 2016) before catching the 2022 divergence and ending around $169.
  **Rate differential only** (without FX) would have reached just ≈$129 —
  so the bulk of the return came from JPY depreciation, not the carry itself.

## Extending the project

- Swap the FX frequency from monthly to daily for more granular stats.
- Add transaction costs and funding spreads on top of the raw yield curve.
- Compare against AUD/JPY and NZD/JPY, historically more classic carry pairs.
- Hedge the FX leg with rolling forwards and compare hedged vs. unhedged P&L.
